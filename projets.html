05
Terminé Java · Spring Boot ADP

Jeu à problèmes

Application web de challenge de programmation développée en équipe de trois dans le cadre du BTS SIO SLAM. Les utilisateurs résolvent des problèmes de code dans un éditeur en ligne, contre la montre, sous l'œil d'un juge qui valide chaque solution. Un système de points et de classement dynamise la compétition entre les participants ; l'historique garde trace de toutes les parties. Quatre rôles distincts structurent l'application : Joueur, Créateur, Juge et Administrateur.

Période 10/2024 → 06/2025 Cadre ADP · BTS SIO SLAM · Équipe de 3 Stack Java · Spring Boot · Oracle DB

06
Terminé Full-Stack ADP

RFTG — Raised From The Graveyard

Plateforme de location de DVDs en ligne réalisée en formation BTS SIO SLAM, articulant trois applications interconnectées autour d'une base MySQL dérivée de Sakila. Luigi (Android / Java) offre aux clients un catalogue filtrable et un système de panier. Mario (Laravel 12 / PHP) fournit au personnel une interface d'administration complète. Toad (Spring Boot 3 / Java 17) orchestre les 67 endpoints sécurisés JWT qui alimentent les deux clients.

Période 10/2025 → 05/2026 Cadre ADP · BTS SIO SLAM Stack Android · Laravel 12 · Spring Boot 3 · MySQL
