Projet en cours : StructurAll'X Web

Dans mon entreprise MTB111, avec mes collègue je travaille sur ce projet. Il vise à lier une application Java à un site web afin de traduire n’importe quel langage informatique en représentation graphique, puis de le retransformer dans un autre langage.

Le but est d’aider à la compréhension et la conversion de structures de code en offrant une visualisation claire et interactive.

Rôle : développement web et intégration de l’API Java.

Projet réalisé : Jeu Problème

Dans le cadre de ma formation, j’ai participé à la réalisation d’un jeu nommé “Jeu Problème”. Ce projet m’a permis de mettre en pratique mes connaissances en programmation et en logique algorithmique.

Le jeu proposait des défis interactifs de type “MOTUS”, encourageant la réflexion et la logique.
